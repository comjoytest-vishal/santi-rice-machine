** project logic-

machine area:
user enter machine code & description to add machine
user can also view, edit, delete existing machine records

work area:
user enter machine code(description), work, notes, staff name to add work
user can also view, edit, delete existing work records

consumable area:
user enter consumable details to add consumable
user can also view, edit, delete existing consumable records

consumable work area:
user enter date, machine code(description), consumable, opening qty, dealer entry, uses qty, staff name, closing qty to add records
user can also view existing consumable records

consumable area (with sub-type):
user enter consumable details to add consumable
user enter sub type of that specific consumable
user can also view, edit, delete existing consumable(with sub-type) records

consumable (with sub-type) work area:
user enter date, machine code(description), consumable, sub types of consumable, opening qty, dealer entry, uses qty, staff name, closing qty to add records
user can also view existing consumable records
